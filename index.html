How to Install Git Bash and push a directory on Mac

In this tutorial you will be shown how to set up and use Git Bash on Mac. This tutorial is may not work on Macs made before 2018 as they used a different terminal shell back then.

1. Install Homebrew

Open a web browser and enter brew.sh in the searchbar or click here. The website should look something like this.

Click the button to the right of the line of text, inidicated in the next image.

This button will copy the code. Now open the terminal and paste with Command + V or rightclick and select "paste".

A whole bunch of text will appear on screen. Eventually it will stop and Homebrew will be installed. Unfortunatly, I already installed Homebrew so I couldn't get an image of the install process. Just know that when the text stops, it should be installed.

2. Install Git Bash

Now that Homebrew is installed, type "brew install git" in the terminal.

This should bring up a whole new load of text. You may have to enter your password for your computer. Once again Git Bash is already installed on my device so I could not capture any images of the process. Once the text stops Git Bash should be installed.

3. Creating a Repo!

Okay! Now that we have Git Bash installed, it's time to make a repo. Go to github.com and click the signup button if you are new or the login button if you already have an account.

Once you are logged in it should look something like this.

Click the indicated button to create a new github repository.

Give your repo a name and an optional description. Leave every other setting how it is. Press "Create repository"

Open the terminal again type "cd " and then drag the folder that you want to add to to your repo into the terminal.

Next, type "git init" and press enter.
